# CONSTRUCT_FILES
This repository contains .c3p files to be imported into the Construct 3 online editor at https://editor.construct.net/.  It also contains standalone code files and audio files in .webm format.  

## Assets Used

### Sprites and Background Images
https://www.kenney.nl/assets/abstract-platformer

https://mikanimus.itch.io/pack-npc-32x32-topdown-rpg

https://cainos.itch.io/pixel-art-top-down-basic

https://dantepixels.itch.io/key-items-16x16


### Audio
